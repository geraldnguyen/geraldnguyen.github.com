The Pull Quote

We need a custom quote.html shortcode and some custom CSS

The quote shortcode:

# quote.html

{{$quote := .Get 0}}

<blockquote class="medium-quote">
  <p>
    {{ markdownify $quote }}
    {{ markdownify .Inner }}
  </p>
</blockquote>
